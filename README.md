android_device_pantech_t100k
============================

CyanogenMod10 device tree for IM-T100K (SKY Vega No.5) - Jellybean Branch

Working:
* RIL
* Graphics
* Sound
* Wifi
* Bluetooth
* Sensors
* GPS
* Camera
* Video

Known issues:
* Can not receive MMS 
* Tethering is not working
* Bluetooth SCO is not working
* HDMI is not tested
* Properiatary softwares are not working ( DMB, etc...)
